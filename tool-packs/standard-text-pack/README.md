# Standard Text Pack

The Standard Text Pack is the first engine-native tool pack for the Local AI Platform.

Included tools:

- `text.clean`
- `text.summarize`
- `text.extract_json`
- `text.classify`
- `text.detect_injection`
- `text.validate_schema`

The first five tools are model-backed and require `model.run`. `text.validate_schema` is deterministic and does not require a model.
